# Nebania ActiveFX
<img src="logo.png" height="200">

ActiveFX is runtime for the link(Script) programming language created by Pilot
<a href="https://github.com/Pilot0253/link-lang#"><p>Project's Page</p></a>

ActiveFX Provides:
<ul>
    <li>LinkScript - a dailect of Link</li>
    <li>ActionObjects - Official framrwork for UI,Graphics,Networking and more</li>
</ul>

<h3>dependices</h3>
<ul>
<li>emscripten</li>
<li>raylib</li>
</ul>

## How to build:

1:get the repository from github:
`git clone https://github.com/tohar777/Nebania-ActiveFX.git`

2.1:compile for emscripten

`make build_wasm'

2.2:compile of a standalone
`make build`
