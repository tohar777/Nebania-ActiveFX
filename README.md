# Nebania ActiveFX
<img src="logo.png" height="270">

ActiveFX is runtime for the link(Script) programming language created by Pilot
<a href="https://github.com/Pilot0253/link-lang#"><p>Project's Page</p></a>

ActiveFX Provides:
<ul>
    <li>ActiveLink - a dailect of Link</li>
    <li>FX Frameworks - Official frameworks for graphics,sound,DOM and everything you need</li>
</ul>

<h3>dependices(that doesn't come with the project!)</h3>
<ul>
<li>SDL2</li>
<li>SDL2_Image</li>

</ul>

## How to build:

1:get the repository from github:
`git clone https://github.com/Nebania/ActiveFX.git`

2.1:compile for emscripten

`make build_wasm'

2.2:compile of a standalone
`make build`
